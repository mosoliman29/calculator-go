# Go Calculator

A simple command-line calculator written in Go.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Division by zero handling
- Interactive CLI
- Clean project structure
- Unit tests

## Project Structure

```
.
├── main.go
├── input.go
├── operations.go
├── operations_test.go
└── go.mod
```

## Run

```bash
go run .
```

## Run Tests

```bash
go test
```

## Example

```
Enter first number: 10
Enter second number: 5

1. Addition (+)
2. Subtraction (-)
3. Multiplication (*)
4. Division (/)

>>> 3

Result: 10 * 5 = 50
```
